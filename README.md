# Travel AI

Travai 白底版静态页面，适合作为旅行规划 AI 产品的前端原型或视觉交付包。

## 快速使用

### 直接打开

双击 `index.html` 即可在浏览器中查看页面。

### 启动本地服务

如果需要更稳定地加载本地资源，可以在项目目录执行：

```bash
python3 -m http.server 8080
```

然后打开：<http://localhost:8080>

项目不需要安装 npm、Node.js 或其他构建依赖。

## 项目结构

- `index.html`：页面结构、样式和交互
- `assets/`：页面图片、插画和本地字体资源
- `logo/`：Travai logo 版本
- `好看的图标-透明/`：透明黑线图标原稿
- `好看的图标-粗线条/`：粗线条插画资源
- `好看的图标-细线条/`：细线条插画资源
- `design-notes/`：视觉和图标设计说明

## 当前页面功能

- Travai 白底版主页面
- 左侧历史对话、我的行程、旅行收藏夹导航
- 搜索框聚焦状态和绿色发送按钮
- 响应式布局，支持桌面和移动端
- 本地透明黑线导航图标

## 维护说明

页面是单文件静态实现。修改布局、颜色或字体时，优先编辑 `index.html` 中的 CSS 变量和对应组件样式。

当前页面英文使用紧凑的 Helvetica Neue 风格字体，中文使用系统中文字体回退；logo 图片保留原始品牌颜色。
