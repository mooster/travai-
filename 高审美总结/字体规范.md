# Travai 字体规范

## 字体变量

当前项目建议使用三类字体变量：

```css
:root {
  --sans: "Travai Sans", "Helvetica Neue", Helvetica, Arial, "Noto Sans SC", "PingFang SC", sans-serif;
  --display: "Avenir Next", "Trebuchet MS", var(--sans);
  --mono: "IBM Plex Mono", "SF Mono", Menlo, ui-monospace, monospace;
  --ui-size: 13px;
}
```

## Display Font

用于：

- Hero 大标题
- 关键品牌标题

规则：

- 首选：`Avenir Next`
- 回退：`Trebuchet MS`
- 字重：`400`
- 字距：`0`
- 不使用负字距。

原因：

`Avenir Next` 更温和、更现代，比强烈衬线字体更适合 Travai 的“理性但可爱”的方向。`Trebuchet MS` 在无授权字体环境下可以作为更圆润的安全回退。

## Sans Font

用于：

- 正文
- 输入框
- 侧栏
- 按钮
- 顶部导航
- 中英文混排

规则：

- 语义名：`Travai Sans`
- 实际回退：`Helvetica Neue`
- 中文回退：`Noto Sans SC`, `PingFang SC`
- 字距：`0`
- 正文字重：`400`
- 导航中文：`700`
- 导航英文：`400`
- UI 字号基准：`13px`

## Mono Font

用于：

- 很小的辅助标签
- 技术提示
- 非主要阅读信息

规则：

- 首选：`IBM Plex Mono`
- 字号通常 `11px - 12px`
- 字距：`0`
- 不要大面积使用。

## 中英文混排

原则：

- 中文承担识别，英文承担补充。
- 中文可加粗，英文保持常规字重。
- 同一行里英文不要突然变大或下沉。
- `AI` 这类短英文需要单独控制 line-height 和 baseline。

推荐写法：

```html
<span class="pink">越用越懂你的 <span class="latin">AI</span></span>
```

```css
.latin {
  font-family: var(--sans);
  font-size: 1em;
  line-height: 1;
}
```

## 当前蓝3字体检查标准

- Hero：`Avenir Next / Trebuchet MS`
- 正文：`Travai Sans / Helvetica Neue`
- 左侧一级导航：中文 `13px/700`，英文 `13px/400`
- 历史记录：默认 `13px/400`，当前项 `13px/700`
- UI 控件：统一接近 `13px`
- 全站字距：`0`

## 不建议

- 不要再引入新的英文字体，除非替换整个系统。
- 不要局部使用衬线字体。
- 不要为了高级感把 UI 字号做得过小。
- 不要让按钮里的英文比中文更重。
