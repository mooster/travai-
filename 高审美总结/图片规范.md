# Travai 图片与图标规范

## 总原则

图片和图标是 Travai 的品牌资产，不是装饰。深色灰蓝底版本里，图片必须低噪音、透明底、边缘干净，不能出现白底、黑边残留或锯齿。

## Logo

### 首选格式

- 优先使用 SVG 矢量图。
- 透明背景。
- 白色线条必须是真正的白色路径，不要从黑色 PNG 后期抠白。
- logo 绿使用当前品牌绿：`#c8e243`。

### PNG 回退

如果必须使用 PNG：

- 至少 2x 显示尺寸。
- 必须是透明背景。
- 边缘不能有黑色残留像素。
- 不要用 CSS filter 把黑色 logo 强行变白。
- 不要把带白底的正方形 logo 直接放进深色底。

### 当前问题记录

`assets/travai-logo-lemon-white.png` 是 PNG，边缘存在深色残留像素，所以在灰蓝底上会看到锯齿或毛边。后续应替换为用户提供的白边 SVG。

## 左侧粗线条图标

用途：

- 历史对话
- 我的行程
- 旅行收藏夹
- 左下角飞机窗

规则：

- 默认：灰白 / 白色透明线条。
- hover：图标自然变成 logo 绿。
- selected：图标和文字一起变成 logo 绿。
- 不用彩色大色块表达状态。
- 图标尺寸建议：`28px`。
- 图标线条要与 logo 的粗线条气质接近。

当前喜好：

深色底上“图标 hover 变 logo 绿”的效果非常符合品牌呼吸感，后续保留为标准交互。

## 飞机窗插图

用途：

- 侧栏底部氛围图。

规则：

- 使用透明图。
- 颜色：灰白低透明度。
- 不要截断。
- 不要抢主导航。
- 当前建议宽度：`210px - 230px`，根据侧栏宽度微调。
- opacity：`0.28 - 0.34`。

## 卷尺线条

用途：

- Hero 大标题上方的轻动效。

规则：

- 颜色：灰白低透明度。
- 不要比标题更亮。
- 不要太粗，避免像蛇或蚯蚓。
- 动效只做轻微展开，不要变成强视觉中心。
- 结尾定位符号也应保持浅灰，不要强行使用高亮色。

## 搜索框与按钮图片

- 搜索框里不放图片符号，保持文字输入感。
- 发送按钮可用线性纸飞机图标。
- 图标 stroke 不宜过粗，和整体细腻的半透明界面匹配。

## 文件存放

建议目录：

- `assets/`：页面实际调用资源。
- `logo/`：logo 原始版本和导出版。
- `好看的图标-透明/`：透明底图标源文件。
- `高审美总结/`：图片规范与偏好记录。

## 导出检查

每次换图前检查：

- 是否透明背景。
- 是否无白底。
- 是否无黑边残留。
- 是否在深色底上边缘干净。
- 是否有足够像素或使用矢量。
- 是否和当前灰蓝底、logo 绿、灰白线条体系一致。
