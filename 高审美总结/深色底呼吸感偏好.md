# Travai 深色底高审美偏好

## 已确认偏好

深色灰蓝底适合保留半透明、低对比、可呼吸的界面质感。用户明确喜欢：左侧图标在 hover 或选中时，从白色/灰白线条自然变成 logo 的柠檬黄绿色。这种变化符合深色底的呼吸感，不刺眼，也能让页面有“活”的状态。

## 视觉原则

- 深色底不要堆太多高亮色。高亮只用于当前状态、主按钮、关键品牌词。
- 图标默认使用白色或灰白透明线条，hover / selected 时变成 logo 绿。
- 选中态不需要大面积色块，优先用文字变色、细竖线、图标 tint、轻微位移。
- 搜索框保留半透明玻璃感，但边框和内层必须克制，避免白框太硬。
- 背景、分割线、卷尺、飞机窗属于氛围层，应比内容低 2-3 个视觉层级。

## 字体体系

- 大标题：`font-family: "Avenir Next", "Trebuchet MS", var(--sans);`
- 大标题气质：温和、现代、有品牌感。
- 正文：`Travai Sans` 语义变量，回退到 `Helvetica Neue / Noto Sans SC / PingFang SC`。
- UI 小字：13px 左右，紧凑、易扫描，字距保持 0。

## 蓝底版状态规则

- 默认图标：灰白、半透明、低存在感。
- hover 图标：logo 柠檬黄绿色，轻微增强亮度。
- selected 图标：同 logo 绿，可配合细线或文字变色。
- 主按钮：logo 柠檬黄绿。
- 次级按钮：透明灰白，不再使用暖黄色或大面积浅色。

## 后续评分时重点检查

- 第一屏是否只让标题和输入框成为主角。
- 左侧是否服务导航，不抢主内容。
- 动效是否轻，只提供状态反馈，不制造噪音。
- 同一层级的英文是否统一使用同一字体、字号和字距。
