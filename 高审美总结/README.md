# Travai 高审美总结

这个目录用于沉淀当前蓝底版的长期设计判断，后续不要只靠截图和口头记忆迭代。

## 文档

- [深色底呼吸感偏好.md](./深色底呼吸感偏好.md)：深色灰蓝底、图标 hover 变 logo 绿、半透明状态的审美偏好。
- [排版系统.md](./排版系统.md)：标题、输入框、导航、提示词、侧栏列表的层级和间距规则。
- [字体规范.md](./字体规范.md)：`Avenir Next / Trebuchet MS / Travai Sans` 的使用边界和中英文混排规则。
- [图片规范.md](./图片规范.md)：logo、粗线条图标、飞机窗、卷尺线条和图片导出要求。

## 当前核心方向

- 深色灰蓝底保持安静，不堆高亮。
- Hero 标题用温和现代的大标题字体。
- 正文和 UI 统一为 Travai Sans 体系。
- 左侧图标默认灰白，hover / selected 变 logo 绿。
- 图片必须透明、干净、无锯齿和黑边残留。
