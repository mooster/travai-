# 粗线条图标配色

以后说“粗线条图标配色”，按这套色温和灰度执行。

- 主线条暖灰：`#7B7770`
- 次级线条灰：`#9B978C`
- 强调深灰：`#585650`
- 冷静分隔线：`#DEDEDB`
- 品牌橙陶土：`#DB6730`
- 正文黑：`#201F1B`
- 页面白底：`#FFFFFF`

使用规则：

- 粗线条插图统一用透明底，不保留棋盘格或整片背景。
- 侧边栏装饰插图优先用主线条暖灰 `#7B7770`。
- 卷尺、辅助路线默认使用浅灰 `#BDBAB4` 的低透明度，避免干扰标题和输入框。
- 卷尺的终点定位符默认保持浅灰，不使用品牌橙色。
- Logo 和核心品牌图形可以保留更重的黑色线条。
- 按钮不使用浅黄色或奶黄色；统一使用低饱和冷灰、灰蓝、灰紫、灰粉、灰绿。
- 侧边栏分隔线不使用暖米色；使用冷静分隔线 `#DEDEDB` 或主线条暖灰的低透明度。
